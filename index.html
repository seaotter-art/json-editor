02 · Replacer

03 · Repeat Detector
반복 출력 감지 & 정리

루프에 빠져 생성한 반복 문장을 자동으로 찾아냅니다. 개별 수정 또는 일괄 삭제로 오염된 컨텍스트를 정리합니다.

최대 3개 파일에서 원하는 인덱스만 체크박스로 선택하고 드래그로 순서를 재배열해 새 JSON으로 추출합니다.

05 · Context Calculator
컨텍스트 사용량 계산 & 분할

현재 사용된 토큰과 최대 컨텍스트를 입력하면 사용률을 계산하고, 초과 시 메시지를 과거백업 / 새시작 두 파일로 분할 다운로드합니다.

도구 주요 목적 파일 수 편집 방식
Inspector
 특정 대화 내용을 찾아 직접 타이핑 수정 1개 검색 → 클릭 → 수동 편집
Replacer
 단어 하나를 파일 전체에서 일괄 치환·제거 1개 Find & Replace (자동 전체)
Repeat Detector
 AI 반복 루프로 생긴 중복 문장 자동 탐색·수정・제거 1개 자동 감지 → 확인 후 수정・삭제
Selected Chat
 여러 파일에서 원하는 인덱스만 골라서 재구성 최대 3개 체크박스 선택 + 드래그 정렬
Context Calculator
 토큰 사용률 확인 후 초과 시 과거백업·새시작으로 분할 1개 토큰 입력 → 자동 계산 → 분할 다운로드
어떤 도구를 써야 할까?
Case 01

대화 중 특정 문장을 찾아 내용을 직접 고쳐야 할 때

→ Inspector 사용
Case 02

잘못 쓴 이름·단어가 파일 전체에 퍼져 있을 때

→ Replacer 사용
Case 03

같은 문장을 반복 출력해 컨텍스트가 오염됐을 때

→ Repeat Detector 사용
Case 04

여러 채팅 파일에서 좋은 대화만 골라 하나로 합칠 때

→ Selected Chat 사용
Case 05

토큰이 컨텍스트 한도를 초과해 채팅을 두 파일로 나눠야 할 때

→ Context Calculator 사용
